# Declaración del Problema de Negocio

## Problema Principal

Cyclistic busca maximizar la rentabilidad del negocio incrementando significativamente el número de membresías anuales. El equipo de finanzas ha confirmado que los **miembros anuales son mucho más rentables** que los usuarios casuales, pero existe una gran oportunidad no aprovechada: **los usuarios casuales ya conocen y utilizan el servicio**.

## Oportunidad de Negocio

En lugar de invertir recursos significativos en la adquisición de nuevos clientes, existe una oportunidad comprobable de **convertir usuarios casuales existentes en miembros anuales**. Estos usuarios ya:

- Conocen el producto y servicio
- Han experimentado los beneficios de Cyclistic
- Han elegido Cyclistic para sus necesidades de movilidad
- Representan una base de clientes con intenciones de compra demostradas

## Pregunta de Investigación Asignada

**¿Cómo usan las bicicletas de Cyclistic de manera diferente los miembros anuales y los usuarios casuales?**

## Impacto Esperado del Análisis

Este análisis debe identificar:

1. **Diferencias comportamentales específicas** entre ambos segmentos
2. **Oportunidades de conversión** basadas en patrones de uso
3. **Segmentos de alta probabilidad** de conversión dentro de usuarios casuales
4. **Estrategias de marketing específicas** para cada segmento identificado
5. **ROI proyectado** de las estrategias propuestas

## Contexto Estratégico

El análisis proporcionará la base para responder las preguntas secundarias:

- **¿Por qué los usuarios casuales comprarían membresías anuales?**
- **¿Cómo puede Cyclistic usar medios digitales para influenciar esta conversión?**

## Análisis Objetivo

- Identificación de diferencias comportamentales
- Explicación de las causas subyacentes de las diferencias
- Propuesta de segmentos específicos de usuarios casuales con alta probabilidad de conversión
- Recomendaciones accionables para marketing

## Criterios de Éxito del Análisis

- Diferencias estadísticamente significativas (p < 0.05) entre segmentos
- Identificación de al menos 3 subsegmentos distintos de usuarios casuales
- Métricas cuantificables de oportunidad de conversión por segmento

## Aplicabilidad Empresarial

- Recomendaciones específicas y accionables
- Estrategias de marketing diferenciadas por segmento
- Proyección de ROI para cada estrategia propuesta

## Rigor Analítico

- Metodología transparente y reproducible
- Documentación completa del proceso analítico
- Visualizaciones profesionales para ejecutivos
