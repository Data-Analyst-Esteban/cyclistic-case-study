# Cyclistic Bike-Share Analysis

**Análisis completo del comportamiento de usuarios para estrategia de conversión de usuarios casuales a membresías anuales**

## Objetivo del Proyecto

Maximizar la rentabilidad de Cyclistic identificando diferencias en patrones de uso entre usuarios casuales y miembros anuales para diseñar estrategias de marketing efectivas que conviertan usuarios casuales en miembros rentables.

## Pregunta de Investigación Principal

**¿Cómo usan las bicicletas de Cyclistic de manera diferente los miembros anuales y los usuarios casuales?**

## Stack Tecnológico

- **Excel/SQL**: Limpieza inicial de datos y consultas básicas
- **Python (Pandas, Scikit-learn)/ R**: Análisis estadístico avanzado y segmentación
- **Tableau/Power BI**: Visualizaciones ejecutivas y dashboards
- **Markdown**: Documentación completa del proceso

## Metodología de Análisis

### Fase 1: Análisis Descriptivo
- Limpieza y preparación de datos (12 meses históricos)
- Estadísticas descriptivas por segmento de usuario
- Identificación de diferencias iniciales

### Fase 2: Segmentación y Clustering
- Agrupamiento de usuarios casuales en subsegmentos
- Análisis de patrones comportamentales
- Identificación de oportunidades de conversión

### Fase 3: Validación Estadística
- Pruebas de hipótesis para confirmar diferencias
- Análisis de significancia estadística
- Modelado predictivo de probabilidad de conversión

### Fase 4: Insights y Recomendaciones
- Traducción de hallazgos a insights de negocio
- Propuestas de segmentación para marketing
- Cálculo de ROI potencial y estrategias

## ROI Proyectado

- **Objetivo**: Identificar segmentos de alta probabilidad de conversión
- **Métrica de éxito**: % de usuarios casuales convertibles por segmento
- **Valor esperado**: Incremento en membresías anuales rentables
